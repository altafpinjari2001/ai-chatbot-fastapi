# AI Chatbot FastAPI

Backend-first AI chatbot service built with FastAPI, designed for clean API integration and extensibility.

[![CI](https://github.com/altafpinjari2001/ai-chatbot-fastapi/actions/workflows/ci.yml/badge.svg)](https://github.com/altafpinjari2001/ai-chatbot-fastapi/actions/workflows/ci.yml)

## What this demonstrates
- API-first LLM service design
- Separation of concerns in backend components
- Deployment-friendly structure

## Key highlights
- FastAPI-based service architecture
- Requirements-managed environment
- CI-enabled quality gate

## Run locally
```bash
pip install -r requirements.txt
# start FastAPI app (see src/ entrypoint)
```
